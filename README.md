The extension package of [Fileformats](https://github.com/arcanaframework/fileformats) provides a library of Python classes types for recognising and handling vendor-specific formats for Canfield files.


## Quick Installation

This extension can be installed for Python 3 using *pip*::

    pip3 install fileformats-vendor-canfield

This will install the core package and any other dependencies
